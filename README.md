# Saleena Cosmatic and Skin Care Website

Single-page, responsive website for **Saleena Cosmatic and Skin Care** in Ambur, built with HTML, CSS and JavaScript.

The site showcases services, classes and quick booking links for WhatsApp, Instagram and Google.

## Files

- `index.html` – main single-page layout (hero, about, services, classes, gallery, contact and footer).
- `styles.css` – styling, layout and responsive behavior.
- `script.js` – mobile navigation, smooth scrolling and booking link handling.
- `README.md` – this file.

## How to Run Locally

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` to open it in your web browser.

No build tools or server are required.

## Booking & Contact Links

Booking buttons (WhatsApp, Instagram, Google) are controlled from `script.js`:

- WhatsApp: `WHATSAPP_URL`
- Instagram: `INSTAGRAM_URL`
- Google: `GOOGLE_URL`

Update these constants if your phone number, Instagram profile or Google Business URL changes.

## Deploy with GitHub Pages

1. Create a new repository in your GitHub account, for example: `saleena-cosmatic-skin-care`.
2. Add all project files (`index.html`, `styles.css`, `script.js`, `README.md`) and push to GitHub.
3. In your repository, go to **Settings → Pages**.
4. Under **Source**, select the `main` branch and **root** (`/`) folder, then save.
5. After a few minutes, your site will be live at:

   `https://<your-username>.github.io/saleena-cosmatic-skin-care/`

Replace `<your-username>` with your GitHub username.
